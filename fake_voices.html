Fake Voice Detection

This project combines cybersecurity and machine learning. I hope that my work on this project will eventually result in my first paper. Specifically, a cybersecurity problem is that computers are becoming increasingly good at generating fake voices. In one example, people were able to swindle lots of money out of a company by mimicking the voice of its CEO using machine learning. Surely, this same technique is applicable in other situations, so we want to limit this problem.

From this problem, I have learned a lot about deep learning in particular. I have used python, tensorflow, and keras for developing the related models.
